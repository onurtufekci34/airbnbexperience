![example](airbnbexample.png)


# Description
This project is a React application created using Vite, a fast build tool for modern web development. It displays a list of Airbnb experience cards and utilizes components such as Navbar, Hero, and Card to render the UI. The data for the cards is fetched from a data file. The project also includes styling using CSS.

# Installation
To run this project locally, follow these steps:

# Clone the repository:

git clone https://github.com/onurtufekci34/airbnbexperience.git

# Navigate to the project directory:

 cd airbnbexperience

# Install the dependencies:

npm install

# Start the development server:

npm run dev

Open your browser and visit http://localhost:3000 to view the application.

# Usage
This project provides a simple and intuitive user interface to view Airbnb experience cards. Upon launching the application, the Navbar component with the Airbnb logo will be displayed at the top. The Hero component showcases a large image, a header, and a description of online experiences. The Card component is used to render multiple cards based on the data fetched from the data file. Each card displays an image, ratings, reviews, location, title, and price.
